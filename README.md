# quicktix

QuickTix by Mystic of Mystic Tech inc.

QuickTix is a very simple, meteor.js based IT ticketing system inspired by Meteor's own todos app.

Functionality to add:
1. Complete User Management portion for admins
2. Ability to add/change ownership of lists by original creators
3. Export Tickets to excel
4. Upload attachments 
5. Upload screenshots and images
6. Notify Ticket creator and contributors of updates via email
7. Enable Edit, Delete functions
8. Enable edit, delete of ticket comments
